–ø—Ä—è–º–æ–π –∏ –Ω–æ—Ä–º–∞–ª—å–Ω–æ–π –ø—Ä—è–º–æ–π –≤ —É–∫–∞–∑–∞–Ω–Ω–æ–π —Ç–æ—á–∫–µ ùë•0. –ò—Å–ø–æ–ª—å–∑—É—è –≥—Ä–∞—Ñ–∏—á–µ—Å–∫–∏–µ

#### –ó–∞–¥–∞–Ω–∏–µ 1

1. –ò—Å–ø–æ–ª—å–∑—É—è –≥—Ä–∞—Ñ–∏—á–µ—Å–∫–∏–µ –ø–∞–∫–µ—Ç—ã Python –ø–æ—Å—Ç—Ä–æ–∏—Ç—å –ø–∞—Ä–∞–º–µ—Ç—Ä–∏—á–µ—Å–∫–∏

#### –ü—Ä–æ–≥—Ä–∞–º–º–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ 1

import matplotlib.pyplot as plt

#—Å–æ–∑–¥–∞–µ–º –º–∞—Å—Å–∏–≤ –∑–Ω–∞—á–µ–Ω–∏—è –ø–∞—Ä–∞–º–µ—Ç—Ä–∞ t

#### –ó–∞–¥–∞–Ω–∏–µ 2

2. –î–ª—è –∑–∞–¥–∞–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ ùëì(ùë•) –∞–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–∏ –Ω–∞–π—Ç–∏ —É—Ä–∞–≤–Ω–µ–Ω–∏—è –∫–∞—Å–∞—Ç–µ–ª—å–Ω–æ–π
–ø—Ä—è–º–æ–π –∏ –Ω–æ—Ä–º–∞–ª—å–Ω–æ–π –ø—Ä—è–º–æ–π –≤ —É–∫–∞–∑–∞–Ω–Ω–æ–π —Ç–æ—á–∫–µ ùë•0. –ò—Å–ø–æ–ª—å–∑—É—è –≥—Ä–∞—Ñ–∏—á–µ—Å–∫–∏–µ
–ø–∞–∫–µ—Ç—ã Python –ø–æ—Å—Ç—Ä–æ–∏—Ç—å –Ω–∞ –æ–¥–Ω–æ–º —Ä–∏—Å—É–Ω–∫–µ –≥—Ä–∞—Ñ–∏–∫ —Ñ—É–Ω–∫—Ü–∏–∏ ùëì(ùë•),
–∫–∞—Å–∞—Ç–µ–ª—å–Ω—É—é –ø—Ä—è–º—É—é, –Ω–æ—Ä–º–∞–ª—å–Ω—É—é –ø—Ä—è–º—É—é –∏ –æ—Ç–º–µ—Ç–∏—Ç—å —Ç–æ—á–∫—É –∫–∞—Å–∞–Ω–∏—è.

$ f(x) = xe^{-x^2} $  
$  x_0 = -1 $

#### –ê–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–æ–µ —Ä–µ—à–µ–Ω–∏–µ 2

–°–Ω–∞—á–∞–ª–∞ –Ω–∞–π–¥–µ–º –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—É—é —Ñ—É–Ω–∫—Ü–∏–∏ f(x):

$ f'(x) = (xe^{-x^2})' = (x)'e^{-x^2} + x(e^{-x^2})' = e^{-x^2}(1-2x^2) $

–¢–µ–ø–µ—Ä—å –Ω–∞–π–¥–µ–º –∑–Ω–∞—á–µ–Ω–∏–µ –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–æ–π –≤ —Ç–æ—á–∫–µ $x_0 = -1$:

$ f'(-1) = e^{-1}(1 - 2(-1)^2) = - e^{-1}$

–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –∫–∞—Å–∞—Ç–µ–ª—å–Ω–æ–π –ø—Ä—è–º–æ–π –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –≤–∏–¥:

$ y = f(x_0) + f'(x_0)(x-x_0) = f(-1) + f'(-1)(x - (-1)) = -e^{-1} + (- \frac{1}{e})(x + 1) =  - \frac{1}{e}(2 + x) $

–£—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–æ—Ä–º–∞–ª—å–Ω–æ–π –ø—Ä—è–º–æ–π –±—É–¥–µ—Ç –ø–µ—Ä–ø–µ–Ω–¥–∏–∫—É–ª—è—Ä–Ω—ã–º –∫–∞—Å–∞—Ç–µ–ª—å–Ω–æ–π –ø—Ä—è–º–æ–π –∏ –ø—Ä–æ—Ö–æ–¥—è—â–∏–º —á–µ—Ä–µ–∑ —Ç–æ—á–∫—É –∫–∞—Å–∞–Ω–∏—è. –¢–∞–∫ –∫–∞–∫ –Ω–∞—Ö–æ–¥–∏–º—Å—è –≤ –¥–≤—É–º–µ—Ä–Ω–æ–º –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ, –ø–µ—Ä–ø–µ–Ω–¥–∏–∫—É–ª—è—Ä–Ω–æ–π –±—É–¥–µ—Ç –ø—Ä—è–º–∞—è, —É –∫–æ—Ç–æ—Ä–æ–π –∫–æ—ç—Ñ—Ñ–∏—Ü–µ–Ω—Ç—ã $ k $ –∏ $ b $ –º–µ–Ω—è—é—Ç—Å—è –º–µ—Å—Ç–∞–º–∏ –∏ –æ–¥–∏–Ω –∏–∑ –Ω–∏—Ö —É–º–Ω–æ–∂–∞–µ—Ç—Å—è –Ω–∞ $ -1 $. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–æ—Ä–º–∞–ª—å–Ω–æ–π –ø—Ä—è–º–æ–π –∏–º–µ–µ—Ç –≤–∏–¥:

$ y = f(x_0) - \frac{1}{f'(x_0)}(x-x_0) = \frac{-1}{f'(-1)}(x - (-1)) + f(-1) = \frac{-1}{-e^{-1}}(x + 1) + (-\frac{1}{e}) = ex + e - \frac{1}{e} $


#### –ü—Ä–æ–≥—Ä–∞–º–º–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ 2


```python
import matplotlib.pyplot as plt 
import numpy as np

# –æ–ø—Ä–µ–¥–µ–ª—è–µ–º f(x)
def f(x):
    return x * np.exp(-x**2)
# –æ–ø—Ä–µ–¥–µ–ª—è–µ–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –∫–∞—Å–∞—Ç–µ–ª—å–Ω–æ–π –ø—Ä—è–º–æ–π
def tangent_line(x):
    return -np.exp(-1)*(2+x)
# –æ–ø—Ä–µ–¥–µ–ª—è–µ–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–æ—Ä–º–∞–ª—å–Ω–æ–π –ø—Ä—è–º–æ–π
def normal_line(x):
    return np.exp(1) * x + np.exp(1) - np.exp(-1)

x = np.linspace(-2,2,100)
y = f(x)

plt.plot(x,y, label = 'f(x)')
plt.plot(x, tangent_line(x), label = '–ö–∞—Å–∞—Ç–µ–ª—å–Ω–∞—è –ø—Ä—è–º–∞—è')
plt.plot(x, normal_line(x), label = '–ù–æ—Ä–º–∞–ª—å–Ω–∞—è –ø—Ä—è–º–∞—è')
plt.scatter(-1, f(-1), color = 'r', label = '—Ç–æ—á–∫–∞ –∫–∞—Å–∞–Ω–∏—è')
plt.legend()
plt.xlabel('x')
plt.ylabel('y')
plt.title('–ì—Ä–∞—Ñ–∏–∫ f(x)')
plt.grid(True)
plt.gca().set_aspect('equal')
plt.show()
```
